## Phân rã chức năng

### Bệnh nhân
- Đặt lịch khám online (chọn bác sĩ, ngày giờ)
- Thanh toán đặt cọc
- Xem đơn thuốc / kết quả khám

### Bác sĩ
- Xem lịch khám trong ngày
- Khám bệnh (ghi chẩn đoán)
- Kê đơn thuốc

### Admin
- Quản lý bác sĩ (CRUD)
- Quản lý danh mục chuyên khoa
- Quản lý danh mục thuốc
